# ATM Management System using Python

## Overview

A menu-driven ATM Management System developed using Python and Object-Oriented Programming (OOP) concepts. The application simulates basic ATM operations including PIN creation, balance inquiry, deposits, and withdrawals.

---

## Features

- Create ATM PIN
- Check Account Balance
- Deposit Money
- Withdraw Money
- PIN Verification
- Secure Data using Private Variables
- Menu-Driven Interface

---

## Concepts Used

- Python
- Object-Oriented Programming (OOP)
- Classes & Objects
- Constructors
- Encapsulation
- Static Variables
- Static Methods
- Getter & Setter Methods
